Loading Animation Test

This page tests the automotive-themed loading animation for the Autoreparatie Op Locatie website.

Animation Features:

  • Automotive Theme: Car icon that drives across screen, rotating gears, oil drops
  • Color Scheme: Uses site colors (#1e3a8a primary, #f97316 accent)
  • Smooth Transitions: Fade in/out effects with proper timing
  • Mobile Responsive: Adapts to different screen sizes
  • Accessibility: Respects reduced motion preferences
  • Performance: Hardware-accelerated animations

Implementation Details:

  • CSS animations in animations.css
  • Loading screen covers entire viewport with high z-index
  • Hidden after 800ms when Vue.js mounts and SiteConfig loads
  • Prevents users from seeing Vue template literals ({{config.key}})
